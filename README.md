# Dungeons of the Abyss

Roguelike built with ReactJS utilizing the canvas.

## Run

### NPM

```
$ git clone https://github.com/Nicocchi/Dungeons-Of-The-Abyss.git
$ cd Dungeons-Of-The-Abyss
$ npm install
$ npm start
```

### Yarn

```
$ git clone https://github.com/Nicocchi/Dungeons-Of-The-Abyss.git
$ cd Dungeons-Of-The-Abyss
$ yarn install
$ yarn start
```
